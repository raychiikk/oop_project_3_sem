    Weather Vibes

    Weather Vibes — це сучасний односторінковий веб-застосунок (SPA) для моніторингу погодних умов, побудований на принципах об'єктно-орієнтованого проєктування (OOP) та чистої архітектури.

    Проєкт демонструє використання патернів проєктування в контексті React-застосунку, розділення логіки та представлення, а також повне покриття UML-моделюванням.

    Функціональність

    • Поточна погода: Відображення температури, стану та детальних характеристик.
    • Прогноз (Forecast): Візуалізація температурних трендів за допомогою графіків (Recharts).
    • Менеджер міст: Додавання та видалення улюблених міст (Local State).
    • Статистика: Аналіз погодних умов (співвідношення сонячних/дощових днів).
    • Теми: Підтримка темного та світлого режимів.

    Технологічний стек

    • Core: React 19, JavaScript (ES6+).
    • Styling: Tailwind CSS 4.
    • Charts: Recharts.
    • Testing: Vitest.
    • Modeling: Mermaid.js (Diagram as Code).
    • Documentation: JSDoc (Docdash theme).

    Архітектура та Патерни

    Проєкт спроєктовано з дотриманням принципів SOLID та Layered Architecture.

    1. Патерни проєктування (GoF)

    • Singleton (Одинак): Реалізовано в класі CityManager. Гарантує існування єдиного списку міст для всього застосунку та централізоване управління станом.
    • Factory Method (Фабричний метод): Реалізовано в WeatherFactory. Інкапсулює логіку створення об'єктів погоди (Sunny, Rainy), що дозволяє легко розширювати систему новими типами умов без зміни основного сервісу.
    • Facade (Фасад): Клас WeatherService виступає єдиною точкою входу для UI-компонентів, приховуючи складність внутрішньої логіки.

    2. Структура проєкту (Layers)

    • Presentation Layer: React-компоненти (/components) — відповідають лише за рендеринг UI.
    • Logic Layer: Бізнес-логіка та фабрики (/js/logic).
    • Data Layer: Моделі даних та менеджери стану (/js/data).

    Встановлення та запуск

    1. Клонування репозиторію:

      git clone [https://github.com/raychiikk/WeatherVibes-OOP.git](https://github.com/raychiikk/WeatherVibes-OOP.git)
cd WeatherVibes-OOP

    2. Встановлення залежностей:

      npm install

    3. Запуск у режимі розробки:

      npm run dev

    4. Запуск тестів:

      npm test

    5. Генерація документації:

      npm run docs
